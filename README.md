# LIBAAS 👕✨

Libaas is a modern responsive clothing website created using HTML, CSS, JavaScript, and Bootstrap.

## 🚀 Features

- Responsive UI Design
- Product Categories
- Add to Cart Counter
- Search Products
- Dark Mode Toggle
- Back to Top Button
- Attractive Hover Effects

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
